# Harmful Content Detection AI

This project explores the use of machine learning techniques to detect harmful online content using multimodal data such as text and images.

## Project Goal
The goal is to design an intelligent system that can assist in identifying harmful, offensive, or inappropriate online content.

## Project Structure
- `data/` - raw and processed datasets
- `notebooks/` - experiments and model testing
- `src/` - core Python scripts
- `models/` - trained models
- `results/` - performance outputs
- `app/` - simple dashboard or interface

## Tools and Technologies
- Python
- Machine Learning
- Deep Learning
- GitHub
